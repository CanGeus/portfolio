Introduction

Perkenalkan, saya Aditya Candra Permana, mahasiswa jurusan Teknik Informatika di Universitas Duta Bangsa. Saya memiliki minat yang mendalam dalam pengembangan solusi teknologi inovatif, terutama di bidang Internet of Things (IoT). Selama studi saya, saya telah terlibat dalam berbagai proyek yang mencakup pengembangan web, dengan fokus khusus pada aplikasi IoT. Repositori GitHub ini adalah tempat saya berbagi portofolio proyek-proyek tersebut. Saya selalu bersemangat untuk belajar mengenai hal-hal baru, serta terus mengembangkan keterampilan saya melalui berbagai kompetisi dan seminar.
